# Elixir LiveView Katas Curriculum

Target: 30 Katas taking the learner from Beginner to Expert.

## Section 1: The Basics (State & Events)
1.  **Hello World** (Static render, basic event) [DONE]
2.  **The Counter** (Integer state, click events) [DONE]
3.  **The Mirror** (Text input, `phx-change`, form bindings)
4.  **The Toggler** (Conditional rendering, boolean state, CSS classes)
5.  **Search Filter** (List rendering, filtering data based on input)

## Section 2: Components & Reusability
6.  **Functional Components** (Refactoring UI into `attr` and `slot` components)
7.  **The Live Component** (Stateful components within a page)
8.  **The Modal** (Handling global UI state, JS commands for show/hide)
9.  **Tabs & Navigation** (Patching the URL, `handle_params`)
10. **Pagination** (Query parameters, limiting list data)

## Section 3: Forms & Data
11. **Registration Form** (Changesets, validation, `phx-submit`)
12. **Nested Forms** (Dynamic inputs, input lists)
13. **Image Uploads** (Live uploads, drag and drop)
14. **Auto-Save Note** (Debounced saves, "saving..." states)
15. **Multi-Step Wizard** (Form funnels, keeping state across steps)

## Section 4: Real-time & Collaboration
16. **Notifications** (Flash messages, auto-dismissing toasts)
17. **PubSub Chat** (Broadcasting messages to other users)
18. **Presence List** (Who is online? Tracking user joins/leaves)
19. **Live Cursor** (Tracking mouse positions of other users)
20. **Collaborative Counter** (Global state via GenServer + PubSub)

## Section 5: JavaScript Integration (Hooks)
21. **Infinite Scroll** (JS Hook for scroll detection + server load)
22. **Chart.js Integration** (Pushing data to client-side libraries)
23. **Map Markers** (Leaflet/Google Maps integration)
24. **Local Storage Sync** (Persisting state to browser storage via Hooks)
25. **Keybindings** (Global keyboard shortcuts)

## Section 6: Architecture & Expert Patterns
26. **Async Operations** (`assign_async`, handling slow data)
27. **Streams** (Handling massive lists efficiently)
28. **LiveView Auth** (Securing mounts, session management)
29. **File Drag & Drop** (Advanced upload zones)
30. **The Final Project** (Mini Trello/Kanban board combining everything)
